# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-01-XX

### Added
- Initial release
- Python flight tracker with timezone support
- PowerShell flight tracker suite (6 variations)
- Flight delay tracking
- UTC timezone conversions
- ASCII route map visualization
- Modular FlightLeg class for reusability
- Windows Terminal color theme (GreenYellowFlight)
- Complete documentation (README, QUICKSTART)

### Features
- **4 flights tracked**: ANA 107, 855, 856, 108
- **3 timezones**: PDT, JST, WIB with automatic conversions
- **2 layovers**: 5h 50m and 16h 05m in Tokyo Haneda
- **Multiple output formats**: Detailed, compact, table, map
- **Color-coded output**: Flight status, delays, timezones
- **No external dependencies** for PowerShell version

### Flight Route
- Outbound: SFO → HND → CGK (July 2-3, 2026)
- Return: CGK → HND → SFO (July 19-20, 2026)
